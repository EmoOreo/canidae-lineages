# Canidae: Lineages

A single-player canid genetics, breeding, hybridization,
and lineage simulation featuring extant and extinct
members of the family Canidae.

## Features

- D/R/M/L genetics system
- Realistic breeding mode
- Sandbox breeding mode
- Extinct species reconstruction
- Lineage tracking
- Hybrid stabilization
- NPC breeder ecosystem
