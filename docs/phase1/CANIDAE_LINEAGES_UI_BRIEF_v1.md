# CANIDAE LINEAGES

## UI / UX DESIGN BRIEF v1.0

---

# PROJECT OVERVIEW

Canidae Lineages is a biologically grounded canid genetics, breeding, health, and lineage simulator.

The simulator currently models:

* Domestic Dogs
* Gray Wolves
* Coyotes
* Foxes
* Dire Wolves
* Hybrids
* Extinct reconstructions

The project emphasizes scientific realism while remaining approachable to players.

Current realism target:

* 95% biological realism
* 5% gameplay abstraction

---

# DESIGN GOALS

The interface should:

* Present large amounts of biological data without overwhelming users
* Support both casual and advanced play
* Prioritize readability
* Support future scientific expansion
* Minimize scrolling
* Make animal comparison easy

---

# FUTURE EXPANSION REQUIREMENTS

The design must leave room for:

* Chromosomal abnormalities
* Intersex conditions
* Disorders of sex development
* Mosaicism
* Chimerism
* Advanced developmental disorders
* Population genetics tools
* Conservation breeding programs

The UI should not require major redesign to support these systems.

---

# VISUAL STYLE

Avoid:

* Fantasy styling
* Cartoon styling
* Mobile gacha aesthetics
* Excessive visual clutter

Prefer:

* Scientific software
* Veterinary software
* Modern management dashboards
* Zoo management interfaces
* Research-oriented applications

---

# DELIVERABLES REQUESTED

The UI/UX designer should provide:

1. Information Architecture
2. User Flow Diagrams
3. Low-Fidelity Wireframes
4. Desktop Mockups
5. Tablet Mockups
6. Mobile Mockups
7. Component Library
8. Design System
